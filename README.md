# AloeWorld LSF 360

Protótipo de aplicação web comercial inspirado no modelo de página apresentado pelo cliente, mas com design, textos e funcionalidades originais da AloeWorld.

## Funcionalidades
- Landing page de marketing responsiva
- Programa formativo com 10 módulos
- Calculadora preliminar de materiais e custos
- Planeador de obra com progresso guardado no navegador
- Assistente técnico demonstrativo
- Formulário de pedido de proposta e envio por WhatsApp
- PWA instalável
- Configuração para Vercel

## Publicar na Vercel
1. Descompactar a pasta.
2. Criar um repositório no GitHub e carregar todos os ficheiros.
3. Na Vercel, escolher **Add New > Project** e importar o repositório.
4. Framework Preset: **Other**.
5. Build Command: deixar vazio.
6. Output Directory: deixar vazio.
7. Fazer **Deploy**.

## Personalização
- Logótipo: `assets/logo-aloeworld.svg`
- Cores: início do ficheiro `styles.css`
- Contacto WhatsApp: procurar `351968917391` em `app.js`
