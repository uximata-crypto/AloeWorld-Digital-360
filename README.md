# Aloeworld Safety Plan 360 — versão 1.1

Aplicação web autónoma para criar planos de emergência e evacuação.

## Testar no computador

1. Extraia o ZIP.
2. Abra o ficheiro `index.html` no Chrome, Edge ou Firefox.
3. O programa funciona sem instalação e sem internet.

## Publicar no Vercel

Carregue diretamente os ficheiros da raiz deste ZIP para a raiz do repositório GitHub. O `index.html` já contém o CSS e o JavaScript necessários. No Vercel use **Framework Preset: Other** e não indique Build Command.

## Ficheiros

- `index.html`: aplicação completa e autónoma.
- `vercel.json`: configuração para Vercel.
- `src/`: fontes separadas para desenvolvimento.

## Funções principais

Paredes, divisões, portas, janelas, textos, cotas, rotas, símbolos de segurança, modelos, seleção, deslocação, propriedades, desfazer/refazer, importação de imagens, guardar/abrir projeto, exportação SVG/PNG e impressão em PDF.
